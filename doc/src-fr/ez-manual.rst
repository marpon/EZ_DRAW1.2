===========================
EZ-Draw, pour dessiner en C
===========================

Dans cette documentation, vous découvrirez comment créer en quelques lignes de
C des programmes qui dessinent dans des fenêtres, intéragissent avec l'utilisateur
au clavier ou la souris, font des animations ou encore affichent des images.

Tout est simplifié au maximum et tient sur quelques fichiers. Par pure magie,
vos programmes fonctionneront sur Linux, Mac OS et Windows.

Cette documentation est présentée en deux parties :
  * un tutorial vous présente pas à pas les notions, avec de nombreux exemples ;
  * le manuel de référence détaille toutes les fonctions disponibles.

Pour faciliter la navigation, vous disposez aussi d'un index et d'une boîte de 
recherche rapide (à gauche).

Bonne lecture !
